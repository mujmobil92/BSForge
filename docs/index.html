Installation

Add one line after Bootstrap. That's it.

CDN
<script src="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/BSForge@latest/dist/loader.js" defer></script>
npm / Vite
npm install bsforge

How it works

The loader scans the page and fetches only the components you actually use.

1️⃣
Add the loader

One script tag, once, in your page head.

2️⃣
Write HTML

Use data-bsforge attributes as documented.

3️⃣
Done

Loader fetches only what the page needs.

Usage
<!-- Basic -->
<div data-bsforge="otp"></div>

<!-- Filled variant with placeholder -->
<div data-bsforge="otp"
     data-variant="filled"
     data-placeholder="·"></div>

<!-- Separator + custom char -->
<div data-bsforge="otp"
     data-separator="3"
     data-separator-char="·"></div>

<!-- Masked + auto-clear after 10 s -->
<div data-bsforge="otp"
     data-mask="true"
     data-timeout="10"></div>

<!-- Custom pattern — only odd digits -->
<div data-bsforge="otp"
     data-pattern="^[13579]$"></div>

<!-- Secure-clear — resets N seconds after page load -->
<div data-bsforge="otp"
     data-secure-clear="30"></div>

<!-- onchange callback — every keystroke -->
<div data-bsforge="otp"
     data-onchange="handleChange"></div>
<script>
  function handleChange(value, complete) {
    console.log(value, complete);
  }
</script>

<!-- Callback on complete -->
<div data-bsforge="otp"
     data-oncomplete="verify"></div>
<script>
  function verify(code) {
    console.log('Code:', code);
  }
</script>
JavaScript API
const otp = document.querySelector('.bsforge-otp')._bsforge;

otp.getValue();          // → "1234" if complete, null if not
otp.setValue("5678");   // set programmatically
otp.reset();             // clear all inputs
otp.setError(true);    // red + shake animation
otp.setError(false);   // green = success
otp.disable();           // disable inputs
otp.enable();            // enable inputs
DOM Events
const el = document.getElementById('myOtp');

el.addEventListener('bsforge:change', e => {
  console.log(e.detail.value);    // current value
  console.log(e.detail.complete);  // true when all filled
});

el.addEventListener('bsforge:complete', e => {
  console.log(e.detail.value);    // final code
});

el.addEventListener('bsforge:reset', () => {
  console.log('reset fired');
});

data-disabled true — Fully disabled
data-required true — Validates on form submit
data-autofocus true — Focus first input on load
data-timeout seconds — Auto-clear after completing
data-secure-clear seconds — Auto-clear + disable after page load
data-oncomplete function name — Called with code on complete
data-onchange function name — Called on every keystroke (value, complete)
data-name string otp Hidden input name for forms

Installation

Add one line after Bootstrap. That's it.

CDN
<script src="https://cdn.jsdelivr.net/gh/mujmobil92/BSForge@latest/dist/loader.js" defer></script>
npm / Vite
npm install bsforge

How it works

The loader scans the page and fetches only the components you actually use.

1️⃣
Add the loader

One script tag, once, in your page head.

2️⃣
Write HTML

Use data-bsforge attributes as documented.

3️⃣
Done

Loader fetches only what the page needs.

Usage
<!-- Basic -->
<div data-bsforge="otp"></div>

<!-- Filled variant with placeholder -->
<div data-bsforge="otp"
     data-variant="filled"
     data-placeholder="·"></div>

<!-- Masked + auto-clear after 10 s -->
<div data-bsforge="otp"
     data-mask="true"
     data-timeout="10"></div>

<!-- onchange callback -->
<div data-bsforge="otp"
     data-onchange="handleChange"></div>
<script>
  function handleChange(value, complete) {
    console.log(value, complete);
  }
</script>
JavaScript API
const otp = document.querySelector('.bsforge-otp')._bsforge;

otp.getValue();          // → "1234" if complete, null if not
otp.setValue("5678");   // set programmatically
otp.reset();             // clear all inputs
otp.setError(true);    // red + shake animation
otp.setError(false);   // green = success
otp.disable();           // disable inputs
otp.enable();            // enable inputs

Dropzone

v0.1.0

A beautiful file upload zone. Supports drag & drop, click to browse, Ctrl+V paste, image previews, file type and size validation.

JavaScript API

Usage
<!-- Basic -->
<div data-bsforge="dropzone"></div>

<!-- Images only, max 2 MB, up to 3 files -->
<div data-bsforge="dropzone"
     data-accept=".jpg,.png,.webp"
     data-max-size="2MB"
     data-multiple="true"
     data-max-files="3"></div>

<!-- onchange callback -->
<div data-bsforge="dropzone"
     data-onchange="handleFiles"></div>
<script>
  function handleFiles(files) {
    console.log(files); // Array of File objects
  }
</script>
JavaScript API
const dz = document.querySelector('.bsforge-dropzone')._bsforge;

dz.getFiles();   // → Array of File objects
dz.reset();      // clear all files
dz.disable();    // disable the dropzone
dz.enable();     // enable the dropzone
